# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-07

### Added

- Interactive prompt template (`vm-service-map-prompt.md`)
- 6 built-in themes: midnight, clean-light, nord, solarized, monokai, paper
- Automatic SSH-based service discovery (systemd units, open ports, dependencies)
- Manual service input as alternative to SSH discovery
- Multi-page PDF layout: overview, service cards, dependency map, quick reference
- Color-coded dependency tags by category (framework, database, AI/LLM, HTTP, etc.)
- Dynamic page numbering and consistent footer across all pages
- Setup script for dependency installation
